# PRISM Demo

An application that demos how Apollo and Prisma GraphQL working together in vanilla javascript.

## Install PROD

```bash
git clone https://github.com/heymp/prisma-demo.git
cd prisma-demo
# create .env file and add your own config
cp .env.example .env
# start production
make start-prod
```
